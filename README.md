# 📚 WebApplication1 - Library Management System / ASP.NET MVC Library Management System

Overview: The Library Management System is a web application developed using ASP.NET MVC that helps manage books, students, and borrowing activities in a library. The system allows users to browse books, view book details, and manage borrowed books, while administrators can manage books, students, and borrowing records.

## 📁 Project Structure

## 📌 Description

This project is an **ASP.NET MVC Library Management System**.

### 👤 Account
- `Login.cshtml` – User login page
- `Register.cshtml` – User registration page

### 🛠 Admin
- `Books.cshtml` – Manage books
- `Borrows.cshtml` – Manage borrowing records
- `CreateBook.cshtml` – Add a new book
- `EditBook.cshtml` – Edit book information
- `Students.cshtml` – Manage students
- `EditStudent.cshtml` – Edit student information

### 📚 Books
- `Index.cshtml` – Display list of books
- `Details.cshtml` – Show book details

### 📖 Borrows
- `MyBooks.cshtml` – List of borrowed books by the user

### 🏠 Home
- `Index.cshtml` – Homepage of the system

### ⚙ Shared
- `_Layout.cshtml` – Main layout for pages
- `Error.cshtml` – Error page
